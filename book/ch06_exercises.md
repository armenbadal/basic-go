# Խնդիրներ ու վարժություններ

_դեռ չկարգավորված ու չխմբագրված_

* Բառային վերլուծիչում փոխել թիվը կարդալու մասն այնպես, որ այն ճանաչի `[+-][0-9]*\.?[0-9]+([eE][+-]?[0-9]+)?` կանոնավոր արտահայտությունը։
* Բառային վերլուծիչում տեքստային լիտերալները կարդացող մասն ընդլայնել այնպես, որ ճանաչի տեքստային լիտերալների ներսում հատուկ սիմվոլները, օրինակ՝ `\n`, `\t`, `\"`, `\\` և այլն։
* Բառայի վերլուծչում կատարել փոփոխություններ լեզուն case insensitive դարձնելու համար։
* `REM`-ը մեկնաբանությունների համար։
* `DIM` հրամանն ընդլայնել երկչափ աղյուսակների համար։ Համապատասխան ձևով ընդլայնել զանգված տարրի ինդեքսավորման գործողությունը։
* `INPUT` հրամանին ավելացնել ոչ պարտադիր հուշման հնարավորություն։
* `PRINT` հրամանն ընդլայնել այնպես, որ կարողանա մեկ տողի վրա արտածել մի քանի արժեք։
* `INPUT` և `PRINT` հրամանները հեռացնել լեզվից։ Ներմուծումն ու արտածումը կազմակերպել ներդրված ենթածրագրերի/ֆունկցիաների միջոցով։
* Իրականացնել `FOR` հրամանի իտերատոր տարբերակը։ Այն պետք է կարողանա թվարկել զանգվածի տարրերը։ Օրինակ. `FOR item IN array ...`։
* Իրականացնել ետպայմանով ցիկլի `DO ... LOOP` կառուցվածքը։
* Իրականացնել ֆայլերի հետ աշխատելու ղեկավարող կառուցվածքներ, ներդրված ֆունկցաիներ։ Օրինակ, `OPEN`, `CLOSE`։
